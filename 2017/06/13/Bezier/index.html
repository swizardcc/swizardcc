贝塞尔曲线

贝塞尔曲线推导过程

  • 直线上两点a, b之间插值。

  • 同样空间坐标系中a(x1, y1, z1), b(x2, y2, z2)插值, 等同于Unity中Vector3.Lerp(a, b, t)。

  • 现在增加到三个点a, b, c, 首先分别求出a, b的插值结果e和b, c 的插值结果f，最后对e,f插值

  • 对上面过程进行化简得到二次曲线的公式:
    B(t) = (1-t)²A + 2t(1-t)B + t²C
      = t²(A - 2B + C) + t(-2A + 2B) + A

  • 同样扩展到2次贝塞尔。
    B(t) = (1-t)³A + 3t(1-t)²B + 3t²(1-t)C + t³D
      = t³(-A + 3B - 3C + D) + t²(3A - 6B + 3C) + t(-3A + 3B) + A
